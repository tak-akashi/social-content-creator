## トラブルシューティング

### よくある問題1: テーマが正しく表示されない

**症状**: インストール後、デザインが崩れている

**原因**: キャッシュプラグインとの競合

**解決策**:
```bash
1. キャッシュプラグインを一時停止
2. ブラウザキャッシュをクリア
3. WordPress管理画面でテーマが有効化されているか確認
4. 子テーマが有効化されているか確認
```

### よくある問題2: ブロックエディタが重い

**症状**: エディタの動作が遅い

**原因**: プラグインの競合

**解決策**:
```bash
1. 不要なプラグインを停止
2. SWELL設定 > エディター設定で不要なブロックを非表示
3. ブラウザの拡張機能を一時停止
```

### よくある問題3: 表示速度が遅い

**症状**: ページ読み込みが遅い

**原因**: 画像の最適化不足

**解決策**:
```bash
1. SWELL設定 > 高速化で全ての最適化をON
2. 画像を圧縮（WebP形式推奨）
3. キャッシュプラグイン導入（WP Rocket推奨）
4. CDN利用検討（Cloudflare等）
```
